Develop observer service for sandbox

The sandbox section should allow to register its data into a global service. This service should capture current board state. Current material evaluation and selected depth. This should have a method to fetch the backend to query the best move

For QoL the button execute should be disabled until the request is processed. This could be handled in the service as well.
